# Deployment Guide: Publishing to Internet Computer Mainnet

This guide walks you through deploying Micro Tools Hub to the Internet Computer mainnet (live production environment).

## Prerequisites

Before deploying to mainnet, ensure you have:

- **DFX SDK** installed (version 0.15.0 or later)
- **Node.js** (version 18 or later) and **pnpm** installed
- **Cycles** in your wallet for deployment costs
- **Internet Identity** configured (if using authentication)
